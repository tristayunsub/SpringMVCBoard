## 개발목적
* Web 백엔드 개발자가 되기 위해 가장 보편적인 Spring의 기본기를 다지기 위함입니다.
* 현재는 모든 지식이 완성되지 않은 상태에서 계속 공부 하며 기능을 추가하고 있습니다.
